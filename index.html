Promise Runner

A simple promisifying utility to make functions run in an asynchronous fashion.

This small tool intends to help reduce the amount of Promise statements and also promisify the functions. You need to keep in mind to pass in the resolve parameter to your function and call the function where ever you need the function to go back 

	  
 function f1(resolve){
   setTimeout(function(){
   consol.log(f1');
   resolve('done'); //this is when we need the function to continue to the next function
  },1000);
 }
 
function f2(resolve){
   setTimeout(function(){
   consol.log(f2');
   resolve('done'); 
  },500);
 }

Compare the above code to the below where you use Promises to make functions run in asynchronous manner.

	  
function f1(resolve){
  return new Promise(function(resolve){
    setTimeout(function(){
       consol.log(f1');
        resolve('done'); 
	    },1000);
    })
}
 
function f2(resolve){
  return new Promise(function(resolve){
   setTimeout(function(){
     consol.log(f2');
     resolve('done'); 
    },500);
  })
}

In the above method the Promise statement has to be repeated in every function and it can cause readability issues and it is not necessary.
